Front vs backend

Fontend: The client-side part of an application that focus on presenting stuff to the end-user

Backend: The server-side part of an application that focus onthe behind-the-scenes, logic, computations, and data storage.

3 basic web technologies:

  • HTML: Markup language that describes the layout and content of a static web page.
  • Javascript: Client side scripting language used to add dynamic functionalityto a webpage.
  • CSS: A language used to style a webpage.
